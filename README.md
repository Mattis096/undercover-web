# Undercover – Jeu de société mobile

## Description

Undercover est un jeu de société adapté en version mobile, conçu pour des soirées entre amis ou en famille. Le but du jeu est de démasquer les joueurs cachés sous le rôle d’Undercover ou de Mr. White avant qu’ils ne vous démasquent.

Ce site web présente le jeu, explique les règles, les rôles et propose un lien pour télécharger l’application depuis le Google Play Store.

## Fonctionnalités

Page d’accueil avec présentation du jeu et bouton de téléchargement.
Section “Découvrir Undercover” présentant les caractéristiques principales du jeu.
Section Règles détaillant le déroulement du jeu étape par étape.
Mini-cartes illustratives représentant les rôles (Civils, Undercovers, Mr. White) avec avatars et descriptions.
Carousel interactif pour visualiser les étapes du jeu.
Section “Rôles” détaillant les objectifs de chaque type de joueur.
Footer avec appel à l’action et liens utiles.
Responsive design pour mobile et tablette.

### Technologies utilisées
HTML5 – Structure du site
CSS3 – Styles et animations, responsive design
JavaScript (optionnel) – Contrôle du carousel et animations de révélation
Images – Illustrations, avatars, icônes

## Installation et utilisation

Cloner le projet :
```
git clone https://github.com/ton-utilisateur/undercover.git
```

## Ouvrir le site :

Naviguer vers le dossier du projet
Ouvrir index.html dans un navigateur moderne

## Télécharger l’application mobile :

Via le bouton “Télécharger sur le Play Store” sur la page d’accueil

## Personnalisation

Couleurs et thèmes : Modifiables dans :root du fichier style.css

Images et icônes : Remplacer les fichiers dans le dossier images/

Textes et titres : Modifiables directement dans les balises HTML
